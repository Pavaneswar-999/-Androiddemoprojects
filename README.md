# Premium Monochrome Calculator

A high-end, distraction-free Android calculator built with a focus on premium UI/UX, accessibility, and clean code.

## ✨ Features
- **Premium Design:** Minimalist Black & White aesthetic with high-contrast typography.
- **Snappy Interactions:** Custom spring-based animations and tactile haptic feedback.
- **Smart UX:** Automatic soft keyboard management and focus clearing.
- **Full Accessibility:** Screen reader support with detailed content descriptions.
- **Material 3:** Built using the latest Material Design components and modern Android practices.

## 🛠 Tech Stack
- **Language:** Kotlin
- **UI:** XML with ConstraintLayout & Flow
- **Architecture:** MVVM-ready clean code
- **Components:** Google Material Design 3
- **Android SDK:** Target SDK 37 (Latest)

## 📸 Preview
The app features a sophisticated monochrome interface where the result appears with a smooth "pop" animation only when calculations are performed.
